# Installation Guide

## Xeno Executor
1. Download the latest version of Xeno Executor from the official website.
2. Extract the downloaded file to a desired location.
3. Run `Xeno.exe`.
4. Follow the setup instructions to complete the installation.
5. Open Xeno Executor and log in with your account.

## Delta Mobile
1. Download Delta Mobile from the official website or trusted source.
2. Install the application on your device.
3. Launch Delta Mobile and follow the on-screen instructions to set it up.
4. Make sure to configure the app settings according to your preferences.
