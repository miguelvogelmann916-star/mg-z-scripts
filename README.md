## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/miguelvogelmann916-star/mg-z-scripts.git
   ```
2. Navigate to the directory:
   ```bash
   cd mg-z-scripts
   ```

3. Run the scripts as needed.
